# FinScope Professional v5.2.1 Real Fund Provider

PBR, PHE ve TLY için gerçek fon veri sağlayıcısı.

## Gerekli ayar

Vercel > Project > Settings > Environment Variables alanına:

FONOLOJI_KEY = fon_...

eklenmelidir. Sonra redeploy yapılır.

API anahtarı yoksa fon kartlarında temsili veri gösterilmez; açıkça API anahtarı beklenir.
